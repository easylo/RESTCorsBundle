# RESTCorsBundle


HOW TO USE:
=========================================

for install

```bash

 composer require "easylo/rest-cors-bundle" 1.*

```


Add bundle
```php
#app/AppKernel.php

public function registerBundles()
    {
        $bundles = array(
        ...,
        new Easylo\RESTCorsBundle\EasyloRESTCorsBundle(),
    );
    
    ...
```

setup config

```yaml

#config.yml :

nothing
    
```
